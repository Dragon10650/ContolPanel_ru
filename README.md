# Панель управления iOS
Быстрая команда помогающая быстро настроивать устройства iOS

> [!NOTE]
> **Находится в бета-тесте, могут быть баги и ошибки**

## Некоторые функции
- **Быстрое включение VPN**
- **Настройки связанные с устройством**
- **Просмотр IP-адреса сети**<br>
*И многое другое*

## Функции, которые скоро появятся
- **Сохранение файла MobileGestalt**
- **Запуск скрытых приложений Apple**
- **Получение информации с сайтов**

> [!NOTE]
> Также можете установить пароль с помощью шортката
